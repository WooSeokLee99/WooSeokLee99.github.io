In the course, GPU programming was covered using C++, but my program was originally written in Python. To ensure an accurate comparison, I ported my program to C++ and conducted a performance evaluation. The results showed that in Python, the main process was 142 times faster, but excessive time was spent on host-to-device delay, making the CPU execution faster overall. In contrast, in C++, the program was 3.27 times faster, and the host-to-device delay was significantly reduced, leading to better overall performance.
